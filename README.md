# helix-sig-finder
Using physicochemical signatures for genomic feature detection.

+ Developed for my final project in the Brandeis GPS Bioinformatics program (RBIF 120).

+ Inspired and developed based on work [published by Mishra et al. in 2021](https://doi.org/10.1093/nar/gkab098), as well as follow-up work published by [Sharma et al. in 2025](https://doi.org/10.1039/D4MO00241E). 

+ More documentation to come soon!
